# Example Markup Language

This project uses Markdown for documentation. Here are examples of the formatting used.

## Headers

# H1
## H2
### H3

## Code Blocks

Use triple backticks for code blocks. Specify the language for syntax highlighting.

```python
def hello_world():
    print("Hello Table!")
```

## Inline Code

Use single backticks for `inline code` or variable names like `table`.

## Lists

- Item 1
- Item 2
  - Nested Item

1. Numbered Item 1
2. Numbered Item 2

## Links

[Link to Table Documentation](Table.md)

## Bold and Italic

**Bold Text** or *Italic Text*.

## Tables (Markdown Table)

| Column 1 | Column 2 |
|Data 1|Data 2|

> Blockquotes are also supported.
